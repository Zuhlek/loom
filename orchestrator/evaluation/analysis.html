What these metrics mean

Time

Inference time
Time the model spent generating, summed across assistant turns. The Anthropic-side bill driver.
Elapsed time
Wall-clock from dispatch to return. Includes inference, tool execution, file I/O, and harness overhead.

Tokens

Input tokens
Fresh prompt bytes (not a cache hit). Billed at the full input rate.
Output tokens
Bytes the model generated. Billed at the full output rate.
Cache writes
Prompt prefix bytes stored for later reuse. Billed at ~1.25× input rate (one-time cost).
Cache hits
Bytes read from the prefix cache. Billed at ~0.1× input rate; dominates input cost on long-context multi-turn runs.

Cache

Cache hit-rate
cache_read / (cache_read + cache_creation + input_tokens). Healthy ≥ 80%; warn ≥ 60%; below 60% means prefix drift — something upstream is mutating the cached bytes.
