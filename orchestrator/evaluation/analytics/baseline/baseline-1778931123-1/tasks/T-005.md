# T-005 — Client shell and esbuild build script

## Goal

Produce the static UI artifacts: HTML shell, CSS, and a TypeScript entry
that esbuild bundles to `dist/client/main.js`. The `main.ts` file at this
task is a minimal stub that mounts and renders the list (logic to follow
in T-006/T-007/T-008). The build script copies `index.html` and
`styles.css` to `dist/client/` and bundles `main.ts` to
`dist/client/main.js`.

## Scope

- `app/src/client/index.html`:
  - Bare HTML document with `<link rel="stylesheet" href="/static/styles.css">`
    and `<script type="module" src="/static/main.js"></script>`.
  - Skeleton: header, a `<form id="save-form">` slot, a `<section id="list">`
    slot, an `<output id="form-error">` slot for inline errors.
- `app/src/client/styles.css`:
  - Minimal styling. Respect `prefers-color-scheme: dark` if free; do not
    add a toggle.
- `app/src/client/main.ts`:
  - Mounts the app, calls `bootstrap()` (to be filled in by T-006).
- `app/scripts/build-client.ts`:
  - Uses `esbuild.build({ entryPoints: ['src/client/main.ts'], bundle: true,
    format: 'esm', outfile: 'dist/client/main.js', platform: 'browser',
    target: 'es2022' })`.
  - Copies `src/client/index.html` and `src/client/styles.css` to
    `dist/client/`.
  - Mkdir `-p` `dist/client/` before write.

## Behaviour delivered

- Provides the static assets US-002 AC1 needs (the page the user loads),
  and the build half of US-005 AC1 ("build the client bundle (if needed)").

## Acceptance criteria (derived from satisfied stories)

- US-002 AC1: after the build, `dist/client/index.html` exists and
  references `/static/main.js` and `/static/styles.css`.
- US-005 AC1: `tsx scripts/build-client.ts` exits 0 and produces
  `dist/client/main.js`, `dist/client/index.html`, `dist/client/styles.css`.

## Test sketches

- Invoke the build script in a test; assert the three output files exist.
- Assert `dist/client/index.html` contains the expected script and link tags.
- Assert `dist/client/main.js` is non-empty (basic sanity).

## Notes

- The bundle script tag uses `type="module"` (ESM bundle) so the client
  can use top-level `await` for the initial fetch if T-006 wants it.
- No framework imports allowed. Pure DOM APIs and `fetch`.
