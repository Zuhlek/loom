## Goal

Bind the Express app to `127.0.0.1:3000` and serve `app/public/`
same-origin so that opening `http://localhost:3000/` returns the UI shell.

## Scope

- `src/server/index.ts`:
  - Open the DB (`openDatabase()`), run migrations (`runMigrations(db)`),
    build the app, then `app.listen(3000, '127.0.0.1')`.
  - SIGINT/SIGTERM handlers close the listener and the DB; exit 0.
  - Boot failure logs and exits with code 1.
- Update `src/server/app.ts` to mount static serving:
  - `express.static(path.resolve(__dirname, '../../public'))` (or
    workspace-relative equivalent under ESM `import.meta.url`).
  - Fallback `GET /` → `public/index.html`.

## Acceptance criteria delivered

- US-002 baseline: the page actually loads at
  `http://localhost:3000`.
- US-003 baseline: the page lives at a real same-origin URL so links
  rendered by the bundle resolve correctly. (Link markup itself is in
  T-008/T-009.)
- Constraint: server binds to `127.0.0.1` only; no outbound network
  calls.

## Tests
