## Goal

Author the static HTML shell and stylesheet that the bundle attaches
behaviour to. This is the structural slice — no JS yet.

## Scope

- `app/public/index.html`:
  - `<!doctype html>` ESM-ready document.
  - `<form id="bookmark-form">` with `<input name="title">`,
    `<input name="url" type="url">`, submit button, and an inline error
    region (e.g. `<p id="form-error" role="alert">`).
  - `<ul id="bookmark-list">` (rendered into by the bundle), an
    `<p id="empty-state">No bookmarks yet</p>` element hidden by default,
    and an `<p id="list-error" role="alert">` region.
  - `<link rel="stylesheet" href="/styles.css">`.
  - `<script type="module" src="/app.js"></script>`.
- `app/public/styles.css`:
  - Minimal layout (single column, readable line length, basic spacing).
  - System dark mode falls out via `prefers-color-scheme: dark` where
    convenient (spec allows but does not require).
  - No third-party CSS, no CDN imports.

## Acceptance criteria delivered

- US-002 AC4 (structural): an empty-state element exists for the bundle
  to toggle.
- US-003 AC2 (structural): the list region exists for the bundle to
  populate with anchor tags carrying
  `target="_blank" rel="noopener noreferrer"`.

`npx tsc --noEmit` (no regressions). HTML/CSS lint is informal.
