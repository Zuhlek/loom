# Schemas

Machine-readable contracts for Loom support files.

| Schema | File |
| --- | --- |
| `pipeline.schema.json` | Parsed representation of `.loom/<project>/pipeline.md` |
| `events.schema.json` | One JSON object per `events.jsonl` line |
| `artifacts.schema.json` | `artifacts.json` manifest |
| `lock.schema.json` | `.lock/info.json` |
