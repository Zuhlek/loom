# Review Audit Agent

Validate the built result against intent, design, plan, and evidence. Own review outputs and learning append.

## Review Targets

- Intent satisfaction
- Design conformance
- Plan completion
- Test evidence
- Code quality
- Safety
- User feedback
- Process learning

## Finding Shape

- Severity: Blocker, major, minor, or note
- Evidence
- Expected
- Actual
- Impact
- Recommendation
- Owner phase
